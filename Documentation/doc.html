Installation

The installation is pretty easy, please follow the steps below.

Step 01:

  • Upload all files/folders located inside "Script" folder to your server. (Don't upload Script folder, upload what inside it) as shown below.



  • Open your browser, and go to http://www.YOURSITE.com
  • You will automatically redirect to installation page

Step 02:

  • Please make sure your server meets the following requirements

    • PHP 8.0+
    • MySQLi Extension
    • fileinfo extension
    • cURL
    • allow_url_fopen
    • ignore_user_abort

    ( Anyway, you can check if the requirements are successfully installed while the installation wizard )

  • After that, you must enter database information
  • Once you have wrote the information, please click the install button.
  • After the installation is done, you'll get a page that looks like



  • Now you can login to control panel via http://YOURSITE.com/cplogin

    • Login Username : monster
    • Login Password : 1234
DANGER: After the installation is finished, please make sure to delete the folder "install".

FAQ

  • When is the next update coming up?
    There is no specified date for new updates.
  • Which host provider do you recommend for CloudMonster?
    We recommend you to use contabo.com
  • Can I use this platform in more than one domain?
    With one purchase code you can use it on one domain name. You need to get new license for each domain name, please check Envato Help Page for more information about licenses
  • Why I get 404 when I visit control panel?
    Please make sure that the .htaccess file is uploaded to your server and updated to latest version.
    If you use Nginx in your server, please setup nginx config related to this script . Open the home directory of the script, you should be able to find nginx.config
  • Why I get 500 internal service error?
    This may cuase due folder permission error, or apache issue, please contact your host provider and let them fix it for you.
  • Where Can I change login username and password ?
    You can change login username and password from the account section in settings page (Settings -> Account).
  • Files not upload to cloud drives, why?
    There may be several reasons, but firstly Make sure max_execution_time is increased and ignore_user_abort is enabled in PHP configuration file. Contact me if not yet uploaded.
  • Can I add multiple drive accounts from the same drive type ?
    Yes you can. For example you can add several google drive accounts (or any other)
  • Public file or bucket link redirects are a bit slow, why?
    To speed up it, disable analytics system from settings page (Settings -> general)
  • How to change the number of file upload processes at once ?
    Change Max upload processes from settings page (Settings -> cloud upload)
  • How to change file upload speed to cloud drives ?
    Change chunk size from settings page (Settings -> cloud upload)
  • How long does it take to check if a cloud file exists? ?
    As the default value every 24 hours, however, you can change the File check time on the Settings page (Settings -> Cloud File / Folder).
